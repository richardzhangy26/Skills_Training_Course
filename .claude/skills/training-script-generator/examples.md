# 训练剧本生成示例

本文档包含完整的输入输出示例，展示如何使用训练剧本生成器。

## 示例 1: 投资推介会场景

### 输入文档

```markdown
# 任务目标
通过智能体技术模拟投资推介会场景，锻炼学生的项目路演能力和应对投资人质疑的能力。

# 任务描述
智能体的人设：智能体是一名专业投资人，名叫林睿，性格严肃，有很强的逻辑思维，在航空航天领域有多年的投资经验。

智能体要完成的任务：有一位创业者正在向你路演自己的创业项目，希望得到你的资金支持，你需要与创业者进行交流。

此次路演会按顺序进行以下几个环节：
1. 项目概述 - 创业者会向你简单描述自己的项目，你可以提出一些问题。
2. 市场和财务分析 - 这一阶段，你需要和创业者讨论他的项目的市场前景、竞品和收益预期。
3. 团队背景 - 你需要了解创业团队的学历情况、工作背景、人员分工等。
4. 总结 - 创业者总结自己的项目优势，投资人最后做点评。

# 评价标准
| 评分项 | 描述 | 分值 |
| --- | --- | --- |
| 项目阐述清晰度 | 学生能否清晰、有逻辑地介绍项目核心价值、技术优势和商业模式 | 30 |
| 市场分析专业性 | 学生对目标市场、竞品分析、收益预期的分析是否专业、有数据支撑 | 30 |
| 应对质疑能力 | 学生能否冷静、有理有据地回应投资人的质疑和追问 | 25 |
| 表达与沟通 | 学生的语言表达是否专业、自信，沟通态度是否得体 | 15 |
```

### 配置说明

**阶段设计思路**:
- 共5个阶段，线性流程
- 第1阶段(0轮)自动输出，营造氛围
- 第2-4阶段为核心交互，轮次递增
- 第5阶段为简短总结

**跳转逻辑**:
- 全部为顺序跳转，每个阶段完成后自动进入下一阶段
- 提示词中包含明确的跳转指令

**特殊处理**:
- 所有阶段(除第一个)都启用历史记录
- 输出字数严格控制(100-200字)
- 林睿人设在所有阶段保持一致

# 任务描述
智能体人设：智能体是一位艺术馆讲解员，名叫小美，25岁，性格热情开朗，对中国传统艺术有深入研究。

智能体要完成的任务：引导学生参观虚拟艺术展馆，介绍展品内容，回答学生问题。

展馆包含三个主题馆：
1. 东方艺术主题馆 - 介绍中国画、书法等传统艺术形式
2. 动物乐活主题馆 - 展示以动物为主题的艺术作品
3. 花鸟书法主题馆 - 聚焦花鸟画和书法艺术

学生可以自由选择参观顺序，每个展馆参观完可以选择去其他展馆或结束参观。

# 评价标准
| 评分项 | 描述 | 分值 |
| --- | --- | --- |
| 参观主动性 | 学生是否主动提出问题，展现出对艺术的兴趣 | 40 |
| 理解深度 | 学生对艺术作品的理解是否深入，能否联系文化背景 | 35 |
| 互动质量 | 学生与讲解员的互动是否积极、有效 | 25 |

- 展馆间可相互跳转，也可结束参观
- 每个展馆出发的线中，有一条设为默认(建议按从左到右顺序)

**特殊处理**:
- "展馆选择"阶段的提示词包含"执行优先级"和意图识别
- 跳转指令、condition、stageName 三者完全一致
