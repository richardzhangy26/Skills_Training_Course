# 参考指南

## 答案生成策略

### 1. 档次差异化控制

#### 覆盖率控制

| 档次 | 评分点覆盖率 | 深度要求 | 创新要求 |
|------|-------------|---------|---------|
| 优秀档 | 100% | 深入全面 | 必须有独到见解 |
| 良好档 | 85-95% | 较好但略有不足 | 有一定新意 |
| 中等档 | 70-80% | 基本达标 | 缺乏新意 |
| 及格档 | 55-65% | 浅显 | 无创新 |
| 不及格档 | <50% | 严重缺失 | 错误或混乱 |

#### 语言风格差异化

**优秀档**：
- 使用专业术语
- 句式完整规范
- 逻辑严密，层次分明
- 避免口语化表达

**良好档**：
- 语言规范流畅
- 偶尔使用专业术语
- 逻辑较清晰
- 个别地方略显口语化

**中等档**：
- 语言较为口语化
- 专业术语使用较少或不准确
- 逻辑基本通顺
- 有"我觉得"、"大概"等口语表达

**及格档**：
- 明显口语化
- 表达不够准确
- 逻辑有跳跃
- 句子结构简单

**不及格档**：
- 语言随意
- 表达不清
- 逻辑混乱
- 可能有错别字或语法错误

### 2. 真实性保证策略

#### 避免AI痕迹

**不要出现**：
- "综上所述"、"总而言之"等模板化结尾
- 过于完美的结构
- 生硬的过渡句
- 重复性的总结

**应该出现**：
- 自然的思考痕迹
- 适当的犹豫和修正
- 个人化的表达
- 符合学生身份的用词

#### 学生常见表达特征

**优秀学生的自然特征**：
- 会有自己的思考角度
- 偶尔使用不太常见的例子
- 论证过程有清晰的逻辑链
- 但不追求完美，可能有小的疏漏

**中等学生的典型特征**：
- 喜欢用"我觉得"、"我认为"
- 举例比较常规
- 分析停留在表面
- 有时会跑题但很快回来

**较差学生的典型特征**：
- 理解偏差，答非所问
- 内容空洞，泛泛而谈
- 逻辑跳跃，前后矛盾
- 字数不足或过于简短

### 3. 作业类型适配

#### 策划案类作业

**长度参考**：
- 优秀档：1500-2500字
- 良好档：1200-2000字
- 中等档：800-1500字
- 及格档：500-1000字
- 不及格档：300-800字

**结构要求**：
- 优秀档：结构完整，包含所有必要部分
- 良好档：结构较完整，个别部分简略
- 中等档：基本结构存在，部分缺失
- 及格档：结构不完整，多个部分缺失
- 不及格档：几乎没有结构

#### 论述题类作业

**长度参考**：
- 优秀档：800-1500字
- 良好档：600-1200字
- 中等档：400-800字
- 及格档：300-600字
- 不及格档：200-400字

**论证要求**：
- 优秀档：论点明确，论据充分，论证严密
- 良好档：论点明确，论据较充分
- 中等档：论点基本明确，论据一般
- 及格档：论点不够清晰，论据不足
- 不及格档：论点混乱或缺失

#### 简答题类作业

**长度参考**：
- 优秀档：200-400字/题
- 良好档：150-300字/题
- 中等档：100-200字/题
- 及格档：50-150字/题
- 不及格档：30-100字/题

#### 计算题/证明题类作业

**步骤要求**：
- 优秀档：步骤完整，逻辑清晰，有必要的文字说明
- 良好档：步骤较完整，个别跳步
- 中等档：基本步骤存在，有跳步
- 及格档：步骤不完整，关键步骤缺失
- 不及格档：步骤混乱或只有结果

### 4. 评分标准对齐

#### 维度覆盖策略

对于每个评价维度，根据档次决定覆盖程度：

**优秀档**：每个维度都达到最高要求
**良好档**：大部分维度达到最高要求，个别维度略有不足
**中等档**：各维度基本达标，但缺乏亮点
**及格档**：部分维度达标，部分明显不足
**不及格档**：多个维度不达标

#### 分值分配策略

假设某维度满分20分：

| 档次 | 预期得分 | 得分策略 |
|------|---------|---------|
| 优秀档 | 18-20分 | 完全符合要求，有加分亮点 |
| 良好档 | 15-17分 | 基本符合，小瑕疵 |
| 中等档 | 12-14分 | 基本达标，但平庸 |
| 及格档 | 10-11分 | 勉强达标，有明显不足 |
| 不及格档 | 5-9分 | 严重不达标 |

### 5. 常见陷阱与避免

#### 优秀档常见陷阱

**过度完美**：
- 陷阱：答案过于完美，不像学生写的
- 避免：加入一些自然的思考痕迹，允许小的不完美

**过于学术**：
- 陷阱：语言过于学术化，超出学生水平
- 避免：保持专业但不过度使用学术术语

#### 中等档常见陷阱

**过于差劲**：
- 陷阱：中等档写得像不及格档
- 避免：中等档应该是"平庸但合格"，不是"差劲"

**缺乏区分度**：
- 陷阱：中等档和良好档区分不明显
- 避免：中等档要有明显的"缺乏深度"特征

#### 不及格档常见陷阱

**过于夸张**：
- 陷阱：写得像故意捣乱，不真实
- 避免：不及格档应该是"能力不足"，不是"态度恶劣"

**完全空白**：
- 陷阱：几乎不写内容
- 避免：不及格档也应该有内容，只是内容质量差

### 6. 质量控制检查清单

生成每份答案后，检查以下项目：

#### 内容质量
- [ ] 答案符合该档次的覆盖率要求
- [ ] 语言风格符合该档次特征
- [ ] 内容真实可信，像学生写的
- [ ] 没有AI痕迹（模板化表达）
- [ ] 长度符合作业类型要求

#### 格式规范
- [ ] 文件头部包含档次名称和预期得分
- [ ] 答案内容部分清晰标注
- [ ] 评分点覆盖情况完整
- [ ] 档次特征说明准确

#### 差异化检查
- [ ] 与上一档次的答案有明显区别
- [ ] 与下一档次的答案有明显区别
- [ ] 各维度得分分布合理

### 7. 提示词优化建议

#### 系统提示词要点

```
1. 明确角色：教育评估专家
2. 明确任务：生成特定档次的学生答案
3. 明确约束：覆盖率、语言风格、真实性
4. 明确禁止：模板化表达、AI痕迹、额外内容
```

#### 用户提示词要点

```
1. 提供完整的上下文（作业描述、评分标准）
2. 明确档次要求（分数区间、核心特征）
3. 提供具体的生成指导（覆盖率、语言风格）
4. 强调输出格式要求
```

### 8. 特殊情况处理

#### 评分标准不明确

如果评分标准描述模糊：
1. 根据作业类型推断合理的评分维度
2. 使用通用的评分框架（内容、结构、语言等）
3. 在评分对照表中注明推断依据

#### 档次描述缺失

如果文档中没有档次描述：
1. 使用标准5档制定义
2. 根据总分推断各档分数区间
3. 在生成时补充档次特征描述

#### 作业类型不明确

如果无法确定作业类型：
1. 根据作业描述推断类型
2. 使用通用长度标准
3. 在答案中保持灵活性
