# LLM 评测提示词库

本目录包含 AI 对话质量评测的完整提示词，共 **5 大维度、20 个子维度**，总分 **100 分**。

## 使用说明

每个提示词文件中包含两个占位符，调用前替换为实际内容：

- `${teacherDoc}` — 教师文档（标准答案/教学设计文档）
- `${dialogueText}` — 实际对话记录（JSON 格式）

所有提示词均要求输出严格 JSON 格式，字段说明见各文件末尾。

## 输出格式说明

所有提示词要求模型输出如下 JSON 结构：

```json
{
  "sub_dimension": "维度名称",
  "score": 0,
  "full_score": 10,
  "rating": "合格",
  "score_range": "",
  "judgment_basis": "详细得分理由（Markdown格式）",
  "issues": [
    {
      "description": "具体问题描述",
      "location": "第X轮对话",
      "quote": "对话原文引用",
      "severity": "high|medium|low",
      "impact": "问题影响简述"
    }
  ]
}
```

| 字段 | 类型 | 取值约束 |
|------|------|----------|
| `score` | number | 0 ~ full_score |
| `rating` | string | 优秀 / 良好 / 合格 / 不足 / 较差 |
| `severity` | string | high / medium / low |
