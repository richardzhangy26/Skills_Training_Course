# homework-answer-generator

根据作业批阅文档生成5个不同档次的学生答案样本，用于测试作业批阅系统。

## 功能概述

本Skill读取docx格式的作业批阅文档，分析其中的作业描述、评分标准和档次描述，然后生成5份不同档次（优秀、良好、中等、及格、不及格）的学生答案样本。

## 使用方法

### 基本用法

1. 在Claude Code中引用作业批阅文档：
   ```
   @作业批阅文档.docx 生成5个档次的学生答案
   ```

2. 或直接告诉Claude：
   ```
   根据这个作业批阅文档生成学生答案样本
   ```

### 工作流程

1. **文档转换**：自动将docx转换为markdown格式
2. **内容解析**：提取作业描述、评分标准、档次描述
3. **答案生成**：依次为5个档次生成答案
4. **文件输出**：在文档所在目录创建文件夹，保存所有答案文件

## 输出文件

假设作业名称为"博物馆活动策划案"，将生成以下文件：

```
博物馆活动策划案/
├── 优秀档答案.md
├── 良好档答案.md
├── 中等档答案.md
├── 及格档答案.md
├── 不及格档答案.md
└── 评分对照表.md
```

### 单份答案文件结构

每份答案文件包含：
- 预期得分区间
- 学生答案内容
- 评分点覆盖情况（各维度的预期得分）
- 档次特征说明

### 评分对照表内容

- 作业基本信息
- 各档次预期得分汇总
- 评价维度得分分布表
- 使用说明

## 档次定义

| 档次 | 分数区间 | 覆盖率 | 核心特征 |
|------|---------|--------|---------|
| 优秀档 | 90-100分 | 100% | 全面深入，见解独到，专业严谨 |
| 良好档 | 80-89分 | ~90% | 较好但细节略有欠缺，规范流畅 |
| 中等档 | 70-79分 | ~70% | 基本达标但创意不足，较口语化 |
| 及格档 | 60-69分 | ~60% | 浅显简单，表达不够准确 |
| 不及格档 | <60分 | <50% | 严重缺失或错误，逻辑混乱 |

## 依赖要求

- `convert_docx_to_markdown.py` 脚本需存在于工作目录
- Python环境需安装：python-docx, mammoth

## 注意事项

1. 确保docx文档包含完整的评分标准和档次描述
2. 生成的答案仅供测试使用，不代表真实学生水平
3. 答案长度会根据作业类型自动调整
4. 如需修改答案风格，可编辑生成的markdown文件
