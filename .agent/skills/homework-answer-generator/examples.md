# 使用示例

## 示例1：博物馆活动策划案

### 输入

用户引用了一个名为"博物馆活动策划案作业批阅.docx"的文件，要求生成学生答案样本。

### 文档内容概要

**作业描述**：
- 作业名称：博物馆活动策划案
- 任务要求：策划一个博物馆活动，包括活动主题、目标受众、活动流程、预算等
- 提交要求：提交策划案文档

**评分标准**（简化示例）：

| 评价维度 | 评价内容 | 分值 |
|---------|---------|------|
| 主题创意 | 活动主题新颖性、吸引力 | 20分 |
| 目标分析 | 目标受众分析准确性 | 15分 |
| 流程设计 | 活动流程完整性、可行性 | 25分 |
| 预算规划 | 预算合理性、详细程度 | 20分 |
| 文案表达 | 语言规范性、逻辑性 | 20分 |

**档次描述**：
- 优秀档（90-100分）：主题新颖独特，分析深入全面，流程设计完善，预算详细合理
- 良好档（80-89分）：主题较好，分析较全面，流程较完善，预算较合理
- 中等档（70-79分）：主题一般，分析基本到位，流程基本完整
- 及格档（60-69分）：主题简单，分析浅显，流程有缺陷
- 不及格档（<60分）：主题不明确，分析缺失，流程混乱

### 输出文件

生成文件夹：`博物馆活动策划案/`

通过问卷调查和社交媒体分析，我们发现该群体偏好...

18:00-18:30 签到入场，领取"时光护照"

19:00-20:30 沉浸式观展：专业讲解员化身历史人物

21:30-22:00 闭幕仪式：点亮"时光之树"

**第三阶段：延续（活动后1周）**
- 线上分享活动照片，评选"最佳时光记录者"
- 发放电子纪念证书

### 四、预算规划

| 项目 | 明细 | 金额（元） |
|-----|------|-----------|
| 场地布置 | 灯光、音响、装饰 | 15,000 |
| 人员费用 | 讲解员、演员、志愿者 | 8,000 |
| 宣传费用 | 线上推广、物料制作 | 5,000 |
| 互动物料 | 道具、纪念品、证书 | 4,000 |
| 应急备用 | 不可预见支出 | 3,000 |
| **合计** | | **35,000** |

### 五、风险评估

1. 人流控制：限流200人，分时段入场
2. 文物安全：设置隔离带，配备安保人员
3. 应急预案：准备医疗点，联系附近医院

## 评分点覆盖情况

### 评价维度1：主题创意（20分）
- 预期得分：19-20分
- 覆盖情况：主题"时光印记"具有诗意和想象空间，结合"博物馆之夜"概念，新颖独特

### 评价维度2：目标分析（15分）
- 预期得分：14-15分
- 覆盖情况：详细分析了三个目标群体，包含调研依据和群体特征

### 评价维度3：流程设计（25分）
- 预期得分：24-25分
- 覆盖情况：三阶段设计完整，时间节点清晰，活动形式丰富多样

### 评价维度4：预算规划（20分）
- 预期得分：19-20分
- 覆盖情况：表格详细，分类合理，包含应急备用金

### 评价维度5：文案表达（20分）
- 预期得分：19-20分
- 覆盖情况：语言规范专业，逻辑清晰，结构完整

## 档次特征说明

本答案体现了优秀档的核心特征：
- 主题新颖独特，具有吸引力
- 目标分析深入，有数据支撑
- 流程设计完善，考虑周全
- 预算详细合理，分类清晰
- 文案专业规范，逻辑严密

无明显不足之处。

## 示例2：简答题作业

对于简答题类型的作业，答案长度会相应调整，以简明扼要的方式回答问题，而非长篇大论的策划案。

### 优秀档答案示例

```markdown
# 优秀档答案

## 预期得分：90-100分

## 答案内容

**问题1：简述博物馆的教育功能。**

博物馆的教育功能主要体现在以下几个方面：

1. **知识传播功能**：博物馆通过藏品展示、专题展览等方式，向公众传播历史、艺术、科学等多领域知识，是学校教育的重要补充。

2. **审美培养功能**：通过艺术品的展示和解读，培养公众的审美能力和艺术鉴赏力。

3. **文化传承功能**：博物馆承载着保护和传承文化遗产的使命，通过展览活动让传统文化得以延续。

4. **社会教育功能**：面向不同年龄、职业的公众开展教育活动，促进终身学习。

**问题2：举例说明博物馆如何开展公共教育。****

以故宫博物院为例，其公共教育活动包括：
- 针对青少年的"故宫知识课堂"
- 面向成人的"故宫讲坛"
- 线上数字展览和虚拟导览
- 与学校合作的研学项目

这些活动形式多样，覆盖面广，取得了良好的社会效果。

---

## 评分点覆盖情况
...
```

## 使用建议

1. **批量测试**：建议一次性测试全部5个档次，以全面评估批阅系统
2. **边界测试**：特别关注良好档与中等档、及格档与不及格档的边界判定
3. **评语评估**：不仅关注分数，还要评估系统给出的评语是否有针对性
4. **迭代优化**：根据测试结果调整答案样本，使其更符合实际学生水平
